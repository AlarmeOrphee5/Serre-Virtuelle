Serre Virtuelle

Application desktop développée en C++ / Qt permettant de gérer des tables de culture, des pots et le suivi de plantations dans une serre virtuelle.

Fonctionnalités
Gestion des tables de culture
Création de tables de culture
Activation / désactivation des tables
Duplication de tables
Suppression de tables

Gestion des pots
Affichage des pots sous forme de grille
Sélection d’un pot
Affichage détaillé :
Nom de la plante
Date de plantation
État du pot

Interface
UI sombre inspirée des dashboards SaaS
Cartes dynamiques
Boutons d’état interactifs
Mise à jour visuelle en temps réel
États des pots

Le projet utilise un système d’états personnalisés avec couleurs dynamiques :

Disponible
Planté
Germination
Croissance
Récolte
Problème
Vide
etc...
Technologies utilisées
C++
Qt 6
Qt Widgets
Layouts dynamiques Qt
Stylesheets Qt (QSS)
